# Release v0.1.0-alpha

**Date:** 2026-06-13  
**Tag:** [`v0.1.0-alpha`](https://github.com/naaa-G/OACP/releases/tag/v0.1.0-alpha)  
**Status:** Public alpha — not production-ready

## Summary

First public release of **OACP (Open Agent Collaboration Protocol)**: a multi-agent task-execution stack with a live browser playground, reference HTTP server, TypeScript and Python SDKs, CLI, and LangChain/AutoGen adapters.

## Quick try

```bash
git clone https://github.com/naaa-G/OACP.git
cd OACP && pnpm install && pnpm build
pnpm oacp run "build todo app" --keep-alive
```

## What's included

| Layer    | Deliverable                                                        |
| -------- | ------------------------------------------------------------------ |
| Protocol | JSON Schema messages `v0.1`, identity, validation                  |
| Runtime  | In-process bus, agent runtime, capability routing                  |
| Server   | HTTP reference server, registry, memory, graph, traces, playground |
| CLI      | `oacp run`, `oacp trace`, `oacp serve`                             |
| SDKs     | `@oacp/sdk`, `oacp-sdk`                                            |
| Demos    | Demo v1/v2, Autonomous Startup Team, example gallery               |
| Adapters | `@oacp/integration-langchain`, `oacp-autogen`                      |
| Docs     | VitePress site + launch kit                                        |

## Breaking change policy

During alpha (`0.1.x`), protocol schemas and public APIs may change without a major bump. Pin git SHAs or exact package versions for experiments.
