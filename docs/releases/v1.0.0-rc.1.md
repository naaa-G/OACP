# Release v1.0.0-rc.1

**Status:** Release candidate — joint launch with MCPLab on Day 60  
**Tag:** `v1.0.0-rc.1` (OACP + MCPLab)  
**Protocol:** `1.0` (frozen Day 54)

## Summary

First release candidate for OACP v1.0.0. Includes the full platform (Docker, API key auth, SQLite persistence, MCPLab↔OACP sync), **OACP Console** as the observability product, and the **Day 58 adoption kit** (bring-your-own-agents, MCP tools server, Cursor skills).

This RC is suitable for:

- Internal / partner dry-runs
- MCPLab gold eval against v1 server
- Final sign-off before `v1.0.0` tag on Day 60
